API Reference

All API endpoints are prefixed with /api/ and use Bearer token authentication (Sanctum).

Authentication

  • POST /auth/register – Register a new user.
  • POST /auth/login – Login and receive a token.
  • POST /auth/forgot-password – Request a password reset.
  • POST /auth/reset-password – Reset password with token.

Forms

  • GET /forms – List user forms.
  • POST /forms – Create a new form.
  • GET /forms/{id} – Get form details.
  • PUT /forms/{id} – Update form.
  • DELETE /forms/{id} – Delete form.
  • POST /forms/{id}/duplicate – Duplicate form.
  • POST /forms/ai-generate – Generate fields via AI.

Submissions

  • GET /submissions – List all submissions.
  • GET /submissions/{id} – Get submission details.
  • DELETE /submissions/{id} – Delete submission.
  • GET /submissions/{id}/export/{format} – Export (csv, excel, pdf).

Webhooks

  • GET /webhooks – List webhooks.
  • POST /webhooks – Create webhook.
  • PUT /webhooks/{id} – Update webhook.
  • DELETE /webhooks/{id} – Delete webhook.
  • POST /webhooks/{id}/test – Test webhook.
